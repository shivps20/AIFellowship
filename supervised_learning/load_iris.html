## Step 8: Generate Confusion Matrix

A confusion matrix is a performance evaluation tool used in classification problems to visualize how well a machine learning model is performing, especially when dealing with multiple classes. Confusion matrix shows where the model makes mistakes. Shows not just how many predictions were correct, but what kind of errors were made.Helps in understanding if a particular class is being confused with another (e.g., Versicolor vs Virginica).

In [26]:
# Generate confusion matrix
